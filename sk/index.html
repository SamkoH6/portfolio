Webová Aplikácia pre Analýzu Kryptomien a Stocks s Predikciou Ceny

Nezávisle som vytvoril projekt analýzy dát a vývoja dashboardu so zameraním na údaje o kryptomenách. Využil som Python na spracovanie údajov, machine learning a vizualizáciu. Dáta som získal pomocou CoinGecko API. Vykonal som spracovanie a analýzu údajov pomocou Pandas, NumPy, Sklearn a TensorFlow. Vytvoril som modely na predpovedanie cien, linear regression a LSTM. Implementované techniky predbežného spracovania údajov, ako je scalling a feature engineering. Vytvoril som user-friendly frontend pomocou Reactu, pomocou MUI a Emotion stylingu pre moderný design a na interaktívne skúmanie údajov.
